# PhysicsHub 团队成员配置
# 在当前会话中通过关键词调用不同角色

## 团队成员

### 1. 文档工程师 @文档工程师
**触发词**: "@文档工程师" 或 "文档工程师"
**职责**: 
- 整理知识点
- 撰写文档
- 公式汇总

**使用方法**:
```
@文档工程师 整理 Ch 13 引力场
@文档工程师 查询 向心力公式
@文档工程师 检查 Ch 12 完整性
```

### 2. 项目助理 @项目助理
**触发词**: "@项目助理" 或 "项目助理"
**职责**:
- 进度跟踪
- 任务管理
- 生成报告

**使用方法**:
```
@项目助理 进度
@项目助理 任务 完成Ch13 @文档工程师
@项目助理 统计
```

### 3. UI工程师 @UI工程师
**触发词**: "@UI工程师" 或 "UI工程师"
**职责**:
- 页面设计
- CSS样式
- 配色方案

**使用方法**:
```
@UI工程师 组件 公式卡片
@UI工程师 配色 警告框
@UI工程师 检查 Ch12样式
```

### 4. 前端工程师 @前端工程师
**触发词**: "@前端工程师" 或 "前端工程师"
**职责**:
- HTML开发
- JS交互
- 组件开发

**使用方法**:
```
@前端工程师 页面 Ch13 theory
@前端工程师 修复 移动端显示
@前端工程师 交互 折叠面板
```

## 使用示例

### 分配任务
```
你: @文档工程师 整理 Ch 13 知识点
你: @UI工程师 设计 Ch 13 页面样式
你: @前端工程师 创建 Ch 13 页面框架
你: @项目助理 任务 完成Ch13 截止3月20日
```

### 查询进度
```
你: @项目助理 进度

回复: 📊 PhysicsHub 进度
━━━━━━━━━━━━━━
整体: 15%
Ch12: ✅ 100%
Ch13: 🔄 20%
```

### 获取设计支持
```
你: @UI工程师 组件 信息框

回复: 🎨 [UI工程师]
```html
<div class="info-box">
  <h4>标题</h4>
  <p>内容</p>
</div>
```
```

## 自动化配置

将以下别名添加到 shell:
```bash
alias doc='echo "@文档工程师"'
alias pm='echo "@项目助理"'
alias ui='echo "@UI工程师"'
alias fe='echo "@前端工程师"'
```
