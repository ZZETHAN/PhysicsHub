# PhysicsHub 飞书多机器人配置指南

## 🤖 4个独立机器人

### 机器人1: 文档工程师
**应用名称**: `PhysicsHub-文档工程师`
**功能**: 知识点整理、文档撰写、公式汇总

**创建步骤**:
1. 访问 https://open.feishu.cn
2. 点击 **"创建企业自建应用"**
3. 填写信息:
   - 应用名称: `PhysicsHub-文档工程师`
   - 应用描述: `A-Level物理学习平台 - 文档与知识库管理`
   - 应用头像: 📚 (上传书本图标)
4. 点击 **"创建"**

**启用机器人**:
- 左侧菜单 → **"机器人"**
- 开启 **"启用机器人"**
- 机器人名称: `@文档工程师`

**权限配置**:
- 左侧菜单 → **"权限管理"**
- 添加权限:
  - `im:chat:readonly` (读取群信息)
  - `im:message:send` (发送消息)
  - `im:message:receive` (接收消息)

**获取凭证**:
- 左侧菜单 → **"凭证与基础信息"**
- 复制:
  - App ID: `cli_xxxxxxxx`
  - App Secret: `xxxxxxxx`

**发布应用**:
- 左侧菜单 → **"版本管理与发布"**
- 点击 **"创建版本"**
- 版本号: `1.0.0`
- 更新说明: `初始版本`
- 点击 **"保存并发布申请"**
