# PhysicsHub 飞书机器人协作团队设计

## 项目概述
**项目名称**: PhysicsHub 学习平台开发
**目标用户**: A-Level物理学生与教师
**核心功能**: 课程资料、学习计划、题库、知识点整理、进度跟踪、资源搜索

## 📱 飞书配置指南

### 创建机器人步骤

1. **进入飞书管理后台** → 应用管理 → 创建应用
2. **基础信息**:
   - 应用名称: PhysicsHub-小P (以此类推)
   - 应用头像: 使用上述设计的风格
   - 应用描述: 对应角色的功能描述

3. **权限配置**:
   - 消息发送权限
   - 群组管理权限
   - 用户信息读取

4. **Webhook配置**:
   - 接收消息事件
   - 配置回调地址

5. **添加到群组**:
   - 将机器人添加到"PhysicsHub开发群"
   - 配置@触发词
