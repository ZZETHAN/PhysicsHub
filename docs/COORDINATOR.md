# PhysicsHub 多机器人协作系统 - 使用指南

## 🎯 架构概览

```
你 (产品经理)
    ↓ 一句话指令
PhysicsHub助手 (协调器)
    ↓ 智能分发
├─ 📚 文档工程师 (并行)
├─ 📊 项目助理 (并行)
├─ 🎨 UI工程师 (并行)
└─ 💻 前端工程师 (并行)
    ↓ 同时完成
汇总结果 → 你
```

## 🚀 快速开始

### 方法一: 使用 ph 命令
```bash
# 进入PhysicsHub目录
cd ~/Desktop/PhysicsHub

# 一句话指令，4个机器人同时工作
./ph "完成Ch 13所有内容"
```

### 方法二: 直接使用Python
```bash
cd ~/Desktop/PhysicsHub
python3 coordinator.py "完成Ch 13所有内容"
```

## 💬 指令示例

### 综合任务（4个机器人同时工作）
```bash
ph "完成Ch 13所有内容"
ph "创建新单元Ch 14"
ph "制作电场学习资料"
```

### 单个机器人任务
```bash
ph "整理Ch 14电场知识点"      # 📚 文档工程师
ph "查看项目进度"             # 📊 项目助理
ph "设计公式卡片组件"         # 🎨 UI工程师
ph "实现可折叠面板"           # 💻 前端工程师
```

### 多个机器人协作
```bash
ph "设计并实现Ch 15组件"      # 🎨UI + 💻前端
ph "整理内容并设计样式"       # 📚文档 + 🎨UI
```

## 📁 文件结构

```
~/Desktop/PhysicsHub/
├── ph                      # 快捷命令
├── coordinator.py          # 协调器主程序
├── config/
│   └── coordinator.yaml    # 4机器人配置
├── scripts/
│   ├── bot_manager.sh      # 飞书机器人管理
│   └── sync_memory.sh      # 记忆同步
└── docs/
    └── COORDINATOR.md      # 本文档
```

## 🔄 共享记忆

4个机器人通过AGFS共享记忆:
```
/Volumes/大硬盘/OpenViking-Agent-Memory/PhysicsHub/
├── shared_memory.json      # 项目进度、任务、规范
├── L0_sessions/            # 工作日志
├── L1_documents/           # 生成的文档
└── L2_knowledge/           # 知识点
```

当一个机器人更新内容时，其他机器人自动看到。

## 🎯 智能分发规则

| 关键词 | 分配机器人 |
|--------|-----------|
| 整理、文档、公式、例题、理论 | 📚 文档工程师 |
| 进度、任务、计划、统计、提醒 | 📊 项目助理 |
| 设计、样式、配色、UI、布局 | 🎨 UI工程师 |
| 开发、代码、交互、功能、修复 | 💻 前端工程师 |
| 完成、创建、制作、生成 | 🎯 全部4个 |

## 🔧 高级用法

### 自定义任务
编辑 `config/coordinator.yaml` 添加新的任务规则。

### 查看机器人状态
```bash
bash scripts/bot_manager.sh status
```

### 同步共享记忆
```bash
bash scripts/sync_memory.sh progress
```
