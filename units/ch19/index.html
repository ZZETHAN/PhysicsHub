Introduction: What is Capacitance?

Real-World Example — Phone Flash:

A phone's flashlight needs instant high power. The battery can't provide this safely. Instead, it slowly charges a capacitor, which releases all energy in milliseconds when the flash fires.

In this unit, you will learn:

  • What capacitance is and how capacitors store energy
  • How capacitors combine in series and parallel circuits
  • Why capacitor discharge follows an exponential pattern
  • Applications from camera flashes to computer motherboards

1. Capacitors and Capacitance

1.1 What is a Capacitor?

A capacitor is a device that stores electric charge and electrical energy. It consists of two conducting plates separated by an insulating material called a dielectric.

Key Structure:
  • Two metal plates (electrodes)
  • An insulating dielectric between the plates (air, paper, ceramic, etc.)
  • When connected to a voltage source, one plate gains +Q and the other −Q
Fig 19.1.1: Basic Capacitor Structure ----------------------------------------------- | | | ++++++++++++++++++++++++++++ (+) | | + dielectric + | | + (insulator) + | | ++++++++++++++++++++++++++++ (-) | | ^ plate A plate B ^ | | | | | | | leads/terminals | | ----------------------------------------------- Two parallel conducting plates separated by an insulating dielectric material.

1.2 How Capacitors Store Charge

When a capacitor is connected to a battery:

  1. Electron Transfer: The negative terminal pushes electrons onto one plate (making it negative)
  2. No Charge Through Dielectric: Crucially, no charge passes through the insulator — it just polarizes
  3. Current Flow: Electrons flow only in the external circuit during charging/discharging
  4. Equilibrium: When PD equals battery e.m.f., charging stops — capacitor holds stored energy
Important Clarification: When we say "charge stored Q", we mean the magnitude of charge on either plate. The total charge on a capacitor is zero (equal +Q and −Q cancel).

V | | | +----------------+ | | | | +-----------+ | | | | | | | C | | | | ___ | | | | | | | | | | |_| | | | +--|--+-----+ | | | | | +-------+--+---------------+ A (ammeter showing current I) Direction of electron flow: clockwise Plate A: negative charge accumulating Plate B: positive charge accumulating ----------------------------------------------- Electrons flow from negative terminal, through external circuit, onto plate A.

1.3 Capacitance: Definition and Formula

Fig 19.1.7: Q-V Graph (Charge vs Voltage) ----------------------------------------------- Q (μC) | | * | * | * | * | * | * +------------------------ V (V) 0 2 4 6 8 10 Gradient = C = Q/V = constant ----------------------------------------------- For a capacitor, charge Q is directly proportional to voltage V. The gradient of this graph equals the capacitance C.
Definition
Capacitance (C) is defined as the ratio of charge stored to the potential difference across the plates: \[ C = \frac{Q}{V} \]

Where: C = capacitance (Farads, F), Q = charge (Coulombs, C), V = potential difference (Volts, V)

From \(C = Q/V\), we can rearrange:
\[ Q = CV \qquad V = \frac{Q}{C} \]
Important Points:
  • 1 Farad is very large; typical capacitors have values in μF, nF, or pF
  • \(1 \, \mu\text{F} = 10^{-6} \, \text{F}\)
  • \(1 \, \text{nF} = 10^{-9} \, \text{F}\)
  • \(1 \, \text{pF} = 10^{-12} \, \text{F}\)

1.4 Capacitance of a Parallel Plate Capacitor

For a parallel plate capacitor with plate area A and plate separation d:

Fig 19.1.8: Parallel Plate Capacitor Dimensions ----------------------------------------------- Plate A (area = A) |==========================| | | <-- dielectric (thickness = d) |==========================| Plate B (area = A) ^ d ^ | | | +-- plate separation +------ plate area Formula: C = ε₀A/d ----------------------------------------------- Key dimensions for a parallel plate capacitor. Plate area A and plate separation d are the key physical parameters.
\[ C = \frac{\varepsilon_0 A}{d} \]

Where: ε₀ = permittivity of free space = \(8.85 \times 10^{-12} \, \text{F m}^{-1}\)

Factors Affecting Capacitance:
  1. Plate Area (A): Larger area → larger capacitance
  2. Plate Separation (d): Smaller separation → larger capacitance
  3. Dielectric Material: Different permittivities affect capacitance

1.5 Energy Stored in a Capacitor

When charging, the power supply does work to separate charges. This work is stored as Electrical Potential Energy.

Key Insight — Capacitors Store Energy, NOT Charge:

A capacitor has equal +Q and −Q (total charge = zero). What makes it useful is the separation of charges, which requires work and stores energy.

Derivation: Work done is variable (voltage goes from 0 to V, average = V/2)

\[ W = \frac{1}{2}QV = \frac{1}{2}CV^2 = \frac{Q^2}{2C} \]

1.6 Capacitance of a Sphere

Surprisingly, all conducting bodies have capacitance! For an isolated conducting sphere of radius r:

\[ C = 4\pi\varepsilon_0 r \]

Example: Earth's capacitance ≈ 710 μF

2. Combination of Capacitors

2.1 Capacitors in Parallel

When capacitors are connected across the same two points (same voltage), they are in parallel.

Fig 19.2.1: Capacitors in Parallel - Circuit Diagram ----------------------------------------------- +------------C1-----------+ | | +------+------+ | | Battery | | | e.m.f. V | | +------+------+ | | | +-----------C2-------------+ Same voltage V across each capacitor Different charges: Q₁ = C₁V, Q₂ = C₂V Total charge: Q = Q₁ + Q₂ ----------------------------------------------- Parallel connection showing same voltage across each capacitor.
Total Capacitance:
\[ C_{total} = C_1 + C_2 + C_3 + ... \]
Key Characteristics:
  1. Same Voltage: All capacitors in parallel have the same PD
  2. Charge Adds: \(Q_{total} = Q_1 + Q_2 + ... = C_1V + C_2V + ...\)
  3. Larger Equivalent: \(C_{total} > C_1\) and \(C_{total} > C_2\)

2.2 Capacitors in Series

When capacitors are connected end-to-end (same charge), they are in series.

Total Capacitance:
\[ \frac{1}{C_{total}} = \frac{1}{C_1} + \frac{1}{C_2} + \frac{1}{C_3} + ... \]

For two capacitors: \(C_{total} = \frac{C_1 C_2}{C_1 + C_2}\)

Key Characteristics:
  1. Same Charge: All capacitors in series store the same charge Q
  2. Voltage Adds: \(V_{total} = V_1 + V_2 + ...\)
  3. Smaller Equivalent: \(C_{total} < C_1\) and \(C_{total} < C_2\)

2.3 Charge and Energy Sharing

When a charged capacitor is connected to an uncharged capacitor:

What Happens?
  • Total charge is conserved (no charge leaves the system)
  • Charge redistributes until both capacitors have the same voltage
  • Energy is LOST! Half the stored energy is dissipated as heat
Key Result: When two capacitors share charge, exactly half of the stored energy is lost (dissipated as heat), regardless of capacitance values.

3. Discharging a Capacitor

3.1 The Discharge Process

Fig 19.3.1: Discharge Circuit Diagram ----------------------------------------------- +--------C--------+ | | | +--+ | | | | = Q | | +--+ | | | | +------+------+----------+------+ | | | R | | +--+--+ | | | | | +------+----+----------------+ | (switch to discharge position) Electron flow: negative plate → R → positive plate Charge decreases: Q → Q - ΔQ → Q - 2ΔQ → ... → 0 ----------------------------------------------- Capacitor discharging through a resistor. Electrons flow through resistor, neutralizing the charge on both plates.

When a charged capacitor is connected across a resistor, it discharges.

Step-by-Step Process:
  1. Electrons flow from negative plate, through resistor, to positive plate
  2. This neutralizes charge on both plates
  3. As charge decreases, PD decreases, current decreases
  4. Eventually: Q = 0, V = 0, current = 0 — capacitor is fully discharged

3.2 Comparison with Radioactive Decay

The discharge of a capacitor is mathematically similar to radioactive decay — both exhibit exponential decay!

Radioactive Decay: \(N = N_0 e^{-\lambda t}\)
Capacitor Discharge: \(Q = Q_0 e^{-t/RC}\)

3.3 Time Constant (τ = RC)

Fig 19.3.4: Time Constant Meaning ----------------------------------------------- Q ↑ Q₀|*** τ=RC *** | * * | * * | * * 0.37Q₀| * * ← Q = Q₀/e at t = τ | * * | * 0.14Q₀| * | * | * 0 |________________________> t 0 τ 2τ 3τ 4τ 5τ ----------------------------------------------- The time constant τ is the time for charge to fall to 37% of its initial value. After 5τ, the capacitor is essentially fully discharged (<1% remaining).
Definition — Time Constant
\[ \tau = RC \]

Where: τ = time constant (seconds, s), R = resistance (Ohms, Ω), C = capacitance (Farads, F)

TimeFraction RemainingFraction Discharged
0100% (Q₀)0%
τ37% (Q₀/e)63%
2τ13.5%86.5%
3τ5.0%95%
5τ0.67%99.33%
After 5 time constants, the capacitor is considered fully discharged (less than 1% of original charge remains).

3.4 Exponential Decay Equations

The charge, voltage, and current on a discharging capacitor all follow exponential decay:

Charge vs Time: \( Q = Q_0 e^{-t/RC} = Q_0 e^{-t/\tau} \)

Voltage vs Time: \( V = V_0 e^{-t/RC} = V_0 e^{-t/\tau} \)

Current vs Time: \( I = I_0 e^{-t/RC} = I_0 e^{-t/\tau} \)

3.5 Applications of Capacitors

Fig 19.3.9: Camera Flash Circuit ----------------------------------------------- Charging phase: Flash phase: Battery ──→ Capacitor Capacitor ──→ Flash (5 seconds) (0.001 seconds) Low current High current Charging Discharging High voltage transformer Trigger circuit ↑ ↓ Battery ──────────────────→ Xenon tube (3V) (300V pulse) ----------------------------------------------- Camera flash uses capacitor to store energy slowly, then release it almost instantaneously as a bright flash.
1. Camera Flash
  • Battery slowly charges capacitor (low current over seconds)
  • When flash triggers, capacitor releases ALL energy in milliseconds
  • Produces extremely high-power flash impossible directly from battery
2. Computer Motherboards
  • Large electrolytic capacitors store charge
  • Smooth out power supply fluctuations
  • Provide extra current during sudden power demands
3. Radio Tuning (Variable Capacitors)
  • Plates can rotate to change effective plate area
  • This changes capacitance, selecting different frequencies
  • Resonance frequency: \( f = \frac{1}{2\pi\sqrt{LC}} \)

Exam Tips and Common Mistakes

Common Mistakes to Avoid

1
Confusing Charge and Energy: Total charge on a capacitor is zero (equal +Q and −Q). What matters for energy is the separation of charges.
2
Series vs Parallel Confusion: Series — same charge, voltage divides. Parallel — same voltage, charge adds. Draw the circuit to visualize before calculating.
3
Unit Conversion Errors: \(1 \, \mu\text{F} = 10^{-6} \, \text{F}\), \(1 \, \text{nF} = 10^{-9} \, \text{F}\), \(1 \, \text{k}\Omega = 1000 \, \Omega\)
4
Energy Loss in Charge Sharing: Many think energy is conserved — Wrong! Half the energy is always lost (dissipated as heat).
5
Time Constant Misunderstanding: τ = RC is NOT the time to fully discharge. After τ, 37% remains. After 5τ, less than 1% remains.
