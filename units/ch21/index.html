Introduction: What is Alternating Current?

What is Alternating Current?
An alternating current (a.c.) is a current that changes direction periodically, and whose magnitude varies with time.

Most power distribution systems use alternating current because it can be easily transformed to different voltages. Understanding AC is essential for studying how electricity is generated, transmitted, and used in everyday life.

1.1 AC vs DC

Definitions

Direct Current (D.C.):
  • Current flows in one direction only
  • Magnitude is constant (or nearly constant)
  • Examples: battery-powered devices
Alternating Current (A.C.):
  • Current changes direction periodically
  • Magnitude varies with time
  • Examples: household mains supply (230V, 50Hz)

Essential Difference:
The essential difference between AC and DC is whether the direction of current changes with time.

1.2 Variation of AC

Key Waveform Parameters

Peak Value ($V_0$ or $I_0$): Maximum magnitude of voltage or current

Period (T): Time for one complete cycle, unit: seconds (s)

Frequency (f): $f = \frac{1}{T}$, unit: hertz (Hz)

Sinusoidal AC

$V = V_0 \sin(\omega t) = V_0 \sin(2\pi ft)$

Where $\omega = 2\pi f$ is the angular frequency.

1.3 Cathode Ray Oscilloscope (CRO)

What is a CRO?

A cathode-ray oscilloscope (CRO) is an electronic device to display waveforms on a fluorescent screen.

CRO Structure

[Image Placeholder]
Fig 21.1.3: CRO Structure

Y-plate: deflects vertically (voltage signal)

X-plate: deflects horizontally (time base/sweep)

Reading from CRO

Peak voltage: $V_0 = \text{vertical divisions} \times \text{Y-gain}$
Period: $T = \text{horizontal divisions} \times \text{time-base}$
Frequency: $f = 1/T$

1.4 Root-Mean-Square (r.m.s.) Values

Why r.m.s.?

The effective value of AC is the value that gives the same heating effect as an equivalent DC current.

r.m.s. Equations

The r.m.s. current is the equivalent direct current that would produce the same heating effect in a resistor.

$I_{r.m.s.} = \frac{I_0}{\sqrt{2}} = 0.707 I_0$
$V_{r.m.s.} = \frac{V_0}{\sqrt{2}} = 0.707 V_0$

Power Calculation

$P = I_{r.m.s.} \times V_{r.m.s.}$
[Image Placeholder]
Fig 21.1.5: r.m.s. Concept

1.5 Average Power in AC Circuits

Instantaneous Power

$P = V_0 I_0 \sin^2(\omega t) = P_0 \sin^2(\omega t)$

where $P_0 = V_0 I_0$

Average Power

$P_{avg} = \frac{P_0}{2} = V_{r.m.s.} \times I_{r.m.s.}$

2.1 Half-Wave Rectification

What is Rectification?

Rectification is converting AC to DC.

Circuit Operation

[Image Placeholder]
Fig 21.2.1: Half-Wave Rectifier Circuit

AC source → Diode → Load resistor R

Positive half-cycle: diode conducts → current flows

Negative half-cycle: diode blocks → no current

Limitation

Half-wave rectification wastes the negative half-cycle — only one half used.

2.2 Bridge Rectifier

Full-Wave (Bridge) Rectification

[Image Placeholder]
Fig 21.2.2: Bridge Rectifier Circuit

4 diodes in bridge configuration

Half-Cycle Diodes Conducting
Positive 2 diodes conduct
Negative 2 diodes conduct
Result: Current flows in same direction for both half-cycles.

Advantage over Half-Wave

  • All input cycles utilized
  • Higher efficiency
  • More power delivered

2.3 Smoothing

The Problem

Rectified output is pulsating DC — it varies with time, not a steady direct voltage. The output rises to a peak then falls to zero, then repeats.

Capacitor Smoothing

[Image Placeholder]
Fig 21.7: Rectified vs Smoothed Output

Capacitor charges at peaks, discharges between peaks to maintain voltage.

Ripple Voltage Factors

The ripple voltage is the small variation between charging cycles.

Factor Effect
Larger C Smaller ripple
Larger R Smaller ripple
Higher frequency Smaller ripple
